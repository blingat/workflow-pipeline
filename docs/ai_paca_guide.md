## 에이징 (14일)
- Day 1~3: 소비자 행동만 (좋아요 20~30, 팔로우 10~15, 댓글 3~5)
- Day 4~7: 수동 발행 1~2개/일 + 이전 활동 유지
- Day 8~14: 발행 3~5개/일 (오픈클로 초안 → 수정 후 직접 발행)
- Day 15~: 완전 자동화 (2H 간격 12개/일)

## 콘텐츠 5대 패턴
1. 무료 대안 ("그거 돈 주고 씀? 무료임") → 리포스트
2. 체크리스트 ("AI로 돈 버는 법 5가지") → 저장
3. 숫자 인증 ("작업시간 87% 줄었음") → 조회수
4. 역발상 도발 ("바이브코딩 한다고 개발자 아님") → 댓글
5. Before/After → 조회수

## 터진 글 기준
- 10분 조회수 500+: 주목 (모니터링)
- 3시간 조회수 2000+: 터짐 → 본계 후보
- 리포스트 50+: 터짐 → 빠코더 톤 재가공

## 금지 사항
- 같은 문구 2번 이상 사용 금지
- 본계와 상호작용 일체 금지
- 글 본문 링크 금지 (프로필 링크만)
- 해시태그 5개 이상 금지 (2개 이하)
- 에이징 전 자동 발행 금지
- 같은 IP에서 본계/서브 동시 로그인 금지
